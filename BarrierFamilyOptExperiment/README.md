# Barrier Family Experiment

`BarrierFamilyMain.py` runs the generic-pipeline barrier study with at least
100 single-underlying configurations and 100 basket configurations.

This is separate from the older dedicated `BarrierMain.py` study.

Generated CSV files are written to `BarrierFamilyOptExperiment/results/` and are
ignored by Git. The committed summary is written to
`Markdown/Barrier/generic_family_results/summary.md`.
